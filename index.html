git - version control system ;

git init - ჩვენ სამუშაო ფოლდერში, შექმნა .git ფოლდერი სადაც სამომავლოდ შეინახავს ჩვენი კოდის ვერსიებს.

git add . - მთელი ჩვენი კოდი დავამატეთ შემდეგ ეტაპზე.

git commit -m "Comment/name of our version" - შევინახეთ კონკრეტული კოდის ვერსია და დაასათაურებს -m -ით.

git log - გვაძლევს სიას ყველა ჩვენს მიერ შექმნილი კომიტის/ვერსიის.

git reset commitID (fe453225443665) --hard - დააბრუნებს ჩვენს კოდს კონკრეტულ კომიტზე/ვერსიაზე.

git status - ამოწმებს გვაქვს თუარა რამე შეცვლილი და დასაკომიტებელი.
